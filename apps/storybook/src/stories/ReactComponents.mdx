# React Components

Inline SVG components for all ISO 7010 safety signs. Each component is tree-shakeable, fully accessible, and accepts the same `SignProps` interface.

## Installation

```sh
npm install @iso-safety-signs/react
```

## Basic usage

Named components are generated by `yarn generate` after running `yarn update`:

```tsx
import { E001EmergencyExit, W001FlammableMaterial } from '@iso-safety-signs/react';

export default function SafetyPanel() {
  return <E001EmergencyExit width={120} />;
}
```

## Dynamic rendering by ID

```tsx
import { SignById } from '@iso-safety-signs/react';

<SignById id="e001-emergency-exit" width={120} />
```

## Props reference — `SignProps`

<Canvas of={ReactStories.EmergencyExit} />
<Controls of={ReactStories.EmergencyExit} />

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `width` | `number \| string` | SVG intrinsic width | Width of the `<svg>` element |
| `height` | `number \| string` | SVG intrinsic height | Height of the `<svg>` element |
| `title` | `string` | Sign name | Accessible title (`<title>` inside SVG) |
| `description` | `string` | Sign description | Accessible description (`<desc>` inside SVG) |
| `className` | `string` | — | CSS class on the root `<span>` wrapper |
| `style` | `React.CSSProperties` | — | Inline styles on the root `<span>` wrapper |
