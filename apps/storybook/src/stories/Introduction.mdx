## What is ISO 7010?

ISO 7010 is the international standard that defines graphical symbols for safety signs used in workplaces, public spaces, and on products worldwide. Signs are grouped into five categories, each with a distinct colour and purpose.

| Category | Code range | Colour | Purpose |
| -------- | ---------- | ------ | ------- |
| Emergency | E001–E076+ | Green | Emergency exits, first aid, eye wash, evacuation |
| Fire equipment | F001–F019+ | Red | Fire extinguisher, hose reel, alarm call point |
| Mandatory | M001–M072+ | Blue | Required actions — PPE, handrails, procedures |
| Prohibition | P001–P081+ | Red/white | Forbidden actions — no smoking, no entry |
| Warning | W001–W089+ | Yellow | Hazard alerts — flammable, high voltage, biohazard |

## Quick start

### React

```sh
npm install @iso-safety-signs/react
```

```tsx
import { SignById } from '@iso-safety-signs/react';

<SignById id="e001-emergency-exit" width={64} />
```

Named components are also generated for each sign:

```tsx
import { E001EmergencyExit, W001FlammableMaterial } from '@iso-safety-signs/react';

<E001EmergencyExit width={64} />
<W001FlammableMaterial width={64} title="Flammable" />
```

### Vue 3

```sh
npm install @iso-safety-signs/vue
```

```vue
<script setup>
import { SignById } from '@iso-safety-signs/vue';
</script>
<template>
  <SignById id="e001-emergency-exit" :width="64" />
</template>
```

### Web Components

```sh
npm install @iso-safety-signs/elements
```

```ts
import { defineCustomElements } from '@iso-safety-signs/elements';
defineCustomElements();
```

```html
<iso-sign sign-id="e001-emergency-exit" width="64"></iso-sign>
<iso-e001-emergency-exit width="64"></iso-e001-emergency-exit>
```

### CSS sprite (no JavaScript)

```sh
npm install @iso-safety-signs/css
```

```js
import '@iso-safety-signs/css/sprite.css';
```

```html
<span class="iso-e001" style="display:inline-block;width:64px;height:64px"
      role="img" aria-label="Emergency exit"></span>
```

### SVG sprite

```sh
npm install @iso-safety-signs/sprite
```

```html
<svg width="64" height="64" role="img" aria-label="Emergency exit">
  <title>Emergency exit</title>
  <use href="/path/to/sprite.svg#e001-emergency-exit" />
</svg>
```

### Programmatic API

```ts
import { getAllSigns, getSign, getSignsByCategory } from '@iso-safety-signs/core';

const allSigns = getAllSigns();                          // Sign[]
const sign     = getSign('e001-emergency-exit');        // Sign | undefined
const warnings = getSignsByCategory('warning');         // Sign[]
```
