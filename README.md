# ISO Safety Signs

Accessible, framework-agnostic packages for all **ISO 7010** workplace safety signs — React components, Vue components, Web Components (Custom Elements), CSS sprite, SVG sprite, and static image assets.

## ISO 7010 Sign Categories

| Category       | Code prefix | Background | Example                      |
| -------------- | ----------- | ---------- | ---------------------------- |
| Emergency      | E001–E068+  | Green      | Emergency exit, first aid    |
| Fire equipment | F001–F017+  | Red        | Fire extinguisher, fire hose |
| Mandatory      | M001–M058+  | Blue       | Wear hard hat, use handrail  |
| Prohibition    | P001–P071+  | Red/white  | No smoking, no entry         |
| Warning        | W001–W073+  | Yellow     | Flammable, high voltage      |

## Quick start

### React

```sh
npm install @iso-safety-signs/react
```

```tsx
import { SignById } from '@iso-safety-signs/react';

export default function SafetyLabel() {
  return <SignById id="e001-emergency-exit" width={120} />;
}
```

Individual named components are generated by `yarn generate` after running `yarn update`:

```tsx
import { E001EmergencyExit, W001FlammableMaterial } from '@iso-safety-signs/react';

<E001EmergencyExit width={64} />
<W001FlammableMaterial width={64} title="Flammable" />
```

### Vue 3

```sh
npm install @iso-safety-signs/vue
```

```vue
<script setup>
import { SignById } from '@iso-safety-signs/vue';
</script>
<template>
  <SignById id="e001-emergency-exit" :width="120" />
</template>
```

### Web Components

```sh
npm install @iso-safety-signs/elements
```

```ts
import { defineCustomElements } from '@iso-safety-signs/elements';
defineCustomElements();
```

```html
<iso-sign sign-id="e001-emergency-exit" width="64"></iso-sign>
<iso-e001-emergency-exit width="64"></iso-e001-emergency-exit>
```

### CSS sprite

```sh
npm install @iso-safety-signs/css
```

```js
import '@iso-safety-signs/css/sprite.css';
```

```html
<span
  class="iso-e001"
  style="display:inline-block;width:64px;height:64px"
  role="img"
  aria-label="Emergency exit"
></span>
```

### SVG sprite

```sh
npm install @iso-safety-signs/sprite
```

```html
<svg width="64" height="64" role="img" aria-label="Emergency exit">
  <title>Emergency exit</title>
  <use href="/path/to/sprite.svg#e001-emergency-exit" />
</svg>
```

### Programmatic API

```ts
import { getAllSigns, getSign, getSignsByCategory } from '@iso-safety-signs/core';

const allSigns = getAllSigns();
const sign = getSign('e001-emergency-exit');
const warnings = getSignsByCategory('warning');
```

## Development

### Prerequisites

- Node.js >= 24 (see `.nvmrc`)
- Yarn 4 (`corepack enable`)

### Setup

```sh
git clone https://github.com/karlnorling/iso-safety-signs.git
cd iso-safety-signs
yarn install
```

### Populate sign data

Scrape Wikipedia, download SVGs from Wikimedia Commons, and generate all source files:

```sh
yarn update
```

This runs three steps in sequence:

1. **scrape** — fetches sign codes, names, and image URLs from the ISO 7010 Wikipedia article
2. **create-assets** — downloads SVGs and converts them to PNG, JPG, and WebP at 240/512/768/1024/2048 px
3. **generate** — generates TypeScript source for all packages

### Compile

```sh
yarn compile
```

### Test

```sh
yarn test
```

### Storybook

```sh
cd apps/storybook && yarn storybook
```
